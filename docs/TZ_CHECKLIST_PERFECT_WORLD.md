# Проверка соответствия ТЗ (Perfect World Board)

## 1. Модели и связи
- `BoardAd`:
  - автор (`User`)
  - заголовок
  - rich-text контент
  - категория из фиксированного списка ТЗ
- `AdResponse`:
  - объявление (`BoardAd`)
  - автор отклика (`User`)
  - текст
  - флаг принятия
- `EmailVerificationCode`:
  - пользователь (`User`)
  - одноразовый код
  - срок действия и статус использования

## 2. Представления и шаблоны
- Список объявлений (`/pw/`) с поиском и фильтром по категории.
- Детальная страница объявления (`/pw/ads/<id>/`) с формой отклика.
- Создание/редактирование/удаление объявления только владельцем.
- Приватная страница откликов (`/pw/responses/`) с фильтром по объявлению.
- Действия владельца объявления: принять или удалить отклик.

## 3. Регистрация и подтверждение email
- Регистрация (`/pw/signup/`) создает неактивного пользователя и код подтверждения.
- Подтверждение (`/pw/verify-email/`) активирует пользователя при верном коде.

## 4. Email-события
- Новый отклик: письмо отправляется автору объявления.
- Принятие отклика: письмо отправляется автору отклика.
- Рассылка: management command `send_pw_newsletter`.

## 5. Категории строго из ТЗ
- Танки
- Хилы
- ДД
- Торговцы
- Гилдмастеры
- Квестгиверы
- Кузнецы
- Кожевники
- Зельевары
- Мастера заклинаний

## 6. Как проверить вручную
1. Зарегистрироваться на `/pw/signup/`.
2. Подтвердить код на `/pw/verify-email/`.
3. Войти и создать объявление на `/pw/ads/create/`.
4. Под другим пользователем оставить отклик на объявление.
5. Войти владельцем объявления, открыть `/pw/responses/`.
6. Принять отклик и проверить письмо у автора отклика.
