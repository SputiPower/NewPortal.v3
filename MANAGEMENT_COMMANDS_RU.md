# Подсказка по командам удаления публикаций

## 1) Удаление новостей по категории
Команда:

```powershell
.\.venv\Scripts\python.exe manage.py delete_news_by_category "Название категории"
```

Что делает:
- Ищет категорию по имени (без учета регистра).
- Находит все публикации типа **новость** (`NW`) в этой категории.
- Перед удалением спрашивает подтверждение в консоли.
- Удаляет только если вы введете `yes`.

## 2) Удаление статей по категории
Команда:

```powershell
.\.venv\Scripts\python.exe manage.py delete_articles_by_category "Название категории"
```

Что делает:
- Ищет категорию по имени (без учета регистра).
- Находит все публикации типа **статья** (`AR`) в этой категории.
- Перед удалением спрашивает подтверждение в консоли.
- Удаляет только если вы введете `yes`.

## Пример подтверждения
После запуска команды в консоли появится вопрос вида:

```text
Введите "yes" для подтверждения:
```

Если ввести что-то кроме `yes`, операция будет отменена.
